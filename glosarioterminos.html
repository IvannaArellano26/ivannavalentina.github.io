GLOSARIO DE TÉRMINOS


    Sistemas y Control:
  • El Sistema Operativo (SO) es el programa principal que administra el hardware, mientras que los Drivers actúan como "traductores" para que el equipo reconozca piezas como impresoras.

    • Hardware Crítico:
  • Memoria RAM: Es la memoria de trabajo rápido; si falla, la PC se vuelve lenta.
  • Disco Duro / SSD: Donde se guarda toda la información de forma permanente.
  • BIOS / UEFI: El primer software que revisa el hardware al encender la computadora.

    Seguridad y Permisos:
  • TCP/IP: Las reglas universales para que Internet funcione.
  • Conflicto de IP:Ocurre cuando dos equipos intentan usar la misma dirección de red.
  • Latencia:El retraso en la conexión; si es alta, el internet se siente lento.

    Uso de la Consola:
  • Base de Conocimiento:Nuestra biblioteca digital para no resolver el mismo problema dos veces.
  • Wireframe:El boceto simple que usamos para planificar el diseño de esta página antes de programarla.
